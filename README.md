## hey, i'm mark

this is my sandbox for minecraft plugin experiments. nothing here is production-grade, but it mostly works.

the honest truth: i write java/kotlin with AI assistance. i describe what i want, claude or chatgpt generates the code, i debug the mess and fix what breaks. rinse and repeat.

i'm not pretending to be a 10x developer here. the minecraft API is huge and i'd rather spend time building features than memorizing every event handler. the AI gets me 80% there, i handle the remaining 20% of actual thinking.

this account exists separately from my main github so i don't clutter it with "vibe-coded" experiments.
